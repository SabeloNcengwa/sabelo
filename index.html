My Projects

Smart Calculators Suite

This project was developed as part of the Internet Programming module under the guidance of Mr. P.K. Ramdeyal. The objective of the assignment was to design and implement a fully functional web-based calculator using core web technologies, specifically HTML, CSS, and JavaScript. We were tasked with replicating the essential features typically found on a physical calculator, including arithmetic operations such as addition, subtraction, multiplication, and division, as well as extended functionalities like clearing input, handling decimal values, and error checking. The project aimed to strengthen our understanding of front-end development principles, event-driven programming, and user interface design. It provided practical experience in writing clean, modular JavaScript code while applying modern styling practices to enhance usability and responsiveness across different devices. This exercise was not only instrumental in reinforcing programming fundamentals, but also allowed us to demonstrate our ability to translate real-world tools into interactive, browser-based applications.
